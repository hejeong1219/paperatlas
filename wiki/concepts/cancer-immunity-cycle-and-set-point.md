# Cancer-Immunity Cycle and Immune Set Point

This concept page links the original stepwise cancer-immunity-cycle model to the later set-point framework that explains why tumors with similar diagnoses can respond very differently to immunotherapy.

## Key Points

- The cancer-immunity cycle is a process model of how antitumor immunity should work.
- The cancer-immune set point is a threshold model of how hard it is for that process to succeed in a given person and tumor.
- Together, they explain why checkpoint blockade can fail for very different reasons: poor antigenicity, failed priming, trafficking failure, immune exclusion, suppressive microenvironment, or dominant inhibitory signaling.
- They also provide the conceptual bridge between biomarker discussions and resistance mechanisms.
