# LLM Wiki

An LLM wiki is a persistent, interlinked markdown knowledge base that is incrementally maintained by an LLM agent as new sources and questions arrive.

## Key Points

- The wiki sits between raw sources and user queries.
- Knowledge is compiled into pages once, then revised over time.
- New sources should update existing pages where appropriate, not just add isolated summaries.
- Queries can produce durable outputs that should sometimes be filed back into the wiki.
- Navigation and maintenance are handled through an index and a chronological log.
