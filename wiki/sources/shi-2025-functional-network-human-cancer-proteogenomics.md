# Mapping the functional network of human cancer through machine learning and pan-cancer proteogenomics

Pan-cancer CPTAC-based study showing that protein-level coexpression can be used to build a high-precision functional network for interpreting cancer proteins and low-frequency drivers.

## Key Points

- Builds `FunMap`, a pan-cancer functional network from proteomics and RNA-seq data across 1,194 individuals spanning 11 cancer types.
- Uses supervised machine learning on harmonized CPTAC-era tumor data to infer functional associations between protein-coding genes.
- Shows that protein coexpression is more functionally informative than RNA coexpression for recovering biologically meaningful relationships in tumors.
- Identifies functional modules linked to cancer hallmarks and clinical phenotypes and helps assign function to understudied cancer-associated proteins.
- Extends proteogenomic interpretation beyond single-marker analysis toward network-level inference in human tumors.

## Relevance

- Relevant to the ptmanchor manuscript because it strengthens the broader claim that protein-level structure contains biologically actionable information that transcriptome-centric approaches can miss.
- Useful as a pan-cancer reference for interpreting abundance-aware signaling context and for situating PTM-aware models inside richer protein-network frameworks.
- Not a phosphosite-correction paper directly, but a strong systems-level companion to kinase and phosphoproteomic inference work.
